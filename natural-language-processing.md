# 자연어 처리

여기서는 Deep-learning 기반 자연어처리 모델에 대해 설명합니다. 

## 용어

#### Model
- 입력을 받아 어떤 처리를 수행하는 함수(Function)입니다. 
- 어떤 입력을 받아서 해당 입력이 특정 범주일 확률을 반환하는 확률함수입니다. 
  - 모델의 출력은 확률입니다.
