# 합성곱 신경망 (CNN_

합성곱 신경망(Convolution Neural Network: CNN)은 합성곱층(Convolutional layer)과 폴링층(Pooling layer)을 포함하는 이미지 처리 성능이 좋은 인공 신경망 알고리즘입니다. 영상 및 사진이 포함된 이미지 데이터에서 객체를 탐색하거나 객체 위치를 찾아내는데 유용한 신경망입니다.
